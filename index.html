A Bit About Me

I'm Sarah, a twentysomething living in Pittsburgh, Pennsylvania. I graduated from Carnegie Mellon University in 2011, where I studied Communication Design and Human-Computer Interaction. I'm currently employed as a User Interface Developer at BarkleyREI, where I develop engaging, responsive web experiences for tourism and higher education clients.

In my free time, I like to learn and try new things, especially when it comes to development. Most recently, I've been spending some time learning CSS Grid and Javascript ES6. To keep up with best practices and trends, I regularly visit development and design blogs and forums, including CSS Tricks, Smashing Magazine, and r/webdev on Reddit.

A newly designed website is in the works. Check back soon for updates!
